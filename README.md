# FS-PREM: Few-Shot Port Risk Event Mining

**Accepted at IEEE ICDM 2026 — Best Paper Runner-Up**

A physics-aware framework for predicting rare port disruption events from streaming hurricane advisories, combining a novel PORTEX disruption index, rolling-window conformal prediction, and lightweight sequential models to deliver calibrated risk estimates in real time under extreme data scarcity.

For the full project writeup — architecture, ML contributions, results tables, PORTEX index definition, and reproduction instructions — see [`fs-prem/README.md`](./fs-prem/README.md).
